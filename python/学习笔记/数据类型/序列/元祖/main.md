## main

## 一、基础

### 1、定义

() 定义元祖

tuple = (1,2,3)

### 2、其他

- 元组可以存放多个不同类型的数据
- 元祖是不可变序列。既创建完后就不用append，insert 方法了，也不能对某项重新赋值
- 元祖不可排序
- 元祖支持不可变序列的全部内置操作
- 如果元祖里面的数据是可变的（比如list），那可以修改这个可变数据。但是不能整体替换这个可变元素 

## 二、常用操作

### 1、创建

```
a = ()
a = tuple()

注意：定义只有一个元素的元祖，需要带上逗号。多余一个元素时不需要
a=(11,)
```



- tuple() 函数，既强制转换
- 将range() , list , 字符串转换为元祖

### 2、删除

```
del 删除整个
```



## 元祖的意义

- 使用不变的对象比较安全，减少错误发生
- 不对的对象自动就是线程安全的，可以省掉处理同步化的开销，可以方便的被共享访问
- 元组在创建时间和占用空间上优于列表
- 元祖能对不需要修改的程序写保护
