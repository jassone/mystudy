## 一、基础

python是强类型语言，数据类型不会因为运算等原因中途改变，**必须显式改变类型**。



```
type() // 获取对象类型
```

## 二、类型

### 1、数字型

### bool

- bool可以和其他类型进行比较，比如数字，字符串等。在比较时，True会转换为1，False会转换为0

- 0被视为false，非0被视为True

  ```
  if 1.1:
    print(1)
  ```
