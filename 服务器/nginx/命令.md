## nginx命令

* nginx -t 
查看nginx的配置文件和状态

* nginx -s reload            重新载入配置文件(不重启)

* nginx -s reopen           重启 Nginx

* nginx -s stop               停止 Nginx

* nginx -s quit
进程完成当前工作后再停止
