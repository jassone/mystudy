## 有pid

```
ss -ntlp | grep nginx // 查看该程序所起的所有端口，linux上可用
ps -x | grep nginx // mac

sudo lsof -i :{{port_number}} | grep LISTEN  
sudo lsof -i :8501 | grep LISTEN  


```



## 无pid

```

sudo ss -tuln | grep ":{{port_number}}"

sudo netstat -tuln | grep ":{{port_number}}"
```



## 查看端口占用

```
sudo lsof -i :7861 -P
```
