### 1、 如何使用凭证

1. **使用SSH密钥验证**:

   在本地生成一对公钥和私钥, 然后将公钥添加到你的Git仓库账户中。

   之后在克隆仓库时,就可以使用SSH协议,而不需要输入用户名和密码。

   - 步骤如下:

     复制

     ```
     # 生成SSH密钥
     ssh-keygen -t rsa -b 4096 -C "your_email@example.com"
     
     # 将公钥添加到你的Git账户
     # 以GitHub为例,登录账户 -> Settings -> SSH and GPG keys -> New SSH key
     
     # 之后克隆仓库
     git clone git@github.com:username/repo.git
     ```

2. **使用Git凭证管理器**: 

   Git有一个凭证管理器,可以存储你的用户名和密码,以便下次自动使用。

   - 在Windows上,可以使用 `git config --global credential.helper store`
   - 在macOS/Linux上,可以使用 `git config --global credential.helper cache`

3. **配置Git的远程仓库URL**:

   - 可以将远程仓库的URL配置为带有用户名的形式,例如: `https://username@github.com/username/repo.git`
   - 这样在克隆时,Git会自动使用配置好的用户名,只需要输入密码即可。
