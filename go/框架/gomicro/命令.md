## 命令



```sh
#创建新服务
micro new helloworld

cd helloworld
make init
go mod vendor
make proto

#启动服务
micro server

#设置为本地环境
micro env set local

#启动你的服务
micro run helloworld

# 检查状态
micro status

# 检查是否已注册
micro services

# 检查日志
micro logs helloworld

```

 



 




# 干掉服务
micro kill helloworld
