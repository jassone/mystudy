## 一些坑

### 1、strings.Split

strings.Split总是会返回一个len=1的空切片

```
	a := ""
	fmt.Println(strings.Split(a, ","))      // [] 
	
	a := "ddd"
	fmt.Println(strings.Split(a, ","))    // [ddd]
```



### 2、strings.TrimLeft

如下, 只要是遇到的字符，都替换掉

```
url := "abcccdefg"
host := "cba"
uri := strings.TrimLeft(url, host)
fmt.Println(uri)   // defg
```

请用TrimPrefix 或者TrimSuffix代替
