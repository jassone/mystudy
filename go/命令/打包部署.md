## 打包部署

### 1、win

```sh
// filename : 文件名
set GOOS=windows 
set GOARCH=amd64 
go build -o "filename"
// 直接filename.exe文件执行即可
```

## 2、linux

```sh
set GOOS=linux 
set GOARCH=amd64 
go build -o "filename"

//或者
env GOOS=linux GOARCH=amd64 go build

上传到linux中赋予执行权限
chmod 777 filename

//项目执行  
nohup ./filename &
```
